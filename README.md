# Treasure-In-The-Sawp

**Introducción**

Este proyecto consiste en el desarrollo de un videojuego multijugador en Unity 3D, donde los jugadores compiten por se el primero en encontrar el tesoro en el mapa en el menor tiempo posible. Para esto el juego implementa mecánicas de movimiento, salto, animaciones, sonidos, interfaz gráfica, entre otros. 


**Desarrollo**
Descripción:
El juego trata sobre dos personajes, que se encuentran como en bosque el cual deben recorrer para encontrar el tesoro, en el menor tiempo posible, el primero en hacerlo es el ganador. 

**Mecánicas:**
**MOVIMIENTO	Personaje 1:** AWSD y para saltar Q.

**Personaje 2:** Flechas y para saltar espacio.

**SALTO**	Uso de la gravedad personalizada.

**CÁMARA** Cada cámara cumple el seguimiento a su jugador asignado. 

**SISTEMA DE VICTORIA**	Detecta al ganador por medio de Triggers.

**TIMER**	Aparece el tiempo en pantalla de cuanto lleva la partida hasta que encuentren el tesoro. 

**SPAWN ALEATORIO**	Las posiciones aleatorias donde puede aparecer el tesoro en cada juego. 

**MENÚ PAUSA**	Tiene las opciones de reanudar, volver al inicio o salir. 

**HERRAMIENTAS UTILIZADAS**	Unity 3D: Para el desarrollo del juego. 

**C#:**  Para la programación.

**TexMeshPro:** UI.

**GitHub:** Para el control de las versiones.
